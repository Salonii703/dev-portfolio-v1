# Dev Portfolio v1

A personal developer portfolio built completely from scratch using vanilla HTML, CSS, and JavaScript — no frameworks.

## 🔗 Live Demo

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)]( https://salonii703.github.io/dev-portfolio-v1/)


## 📌 Project Purpose

This portfolio showcases my skills, tech stack, and projects as a CSE (IoT and Robotics) student. Built as part of an Engineering Foundations assignment focused on production-mindset development — clean code, semantic HTML, proper git history, and real deployment.

## 🚀 Setup Instructions

1. Clone the repository:
```bash
   git clone https://github.com/<your-username>/dev-portfolio-v1.git
```
2. Navigate into the folder:
```bash
   cd dev-portfolio-v1
```
3. Open `index.html` directly in your browser — no build step or server required.
   - Or use VS Code's **Live Server** extension for auto-reload.

## ✨ Features

- Responsive single-page layout (Hero, About, Skills, Projects, Contact)
- Dark/light mode toggle with persistence via `localStorage`
- Semantic HTML5 (`header`, `nav`, `main`, `section`, `footer`)
- Fully external CSS — no inline styles
- Mobile-friendly design with media queries

## 🛠️ Tech Stack

- HTML5
- CSS3 (CSS variables, Flexbox, media queries)
- Vanilla JavaScript (no frameworks/libraries)
